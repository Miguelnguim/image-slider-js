# 🎯 Image Slider (Carousel) en JavaScript

Un image slider moderne, responsive et performant** développé en **JavaScript Vanilla (ES6)** sans framework.

Ce projet met en pratique des concepts essentiels du frontend : manipulation du DOM, animations fluides, gestion des événements et UX moderne.

---

## 🚀 Aperçu

Un carousel élégant avec :

* Navigation fluide
* Animation moderne
* Support mobile (swipe)
* Design épuré

---

## ✨ Fonctionnalités

* ✅ Navigation avec boutons (Next / Previous)
* ✅ Indicateurs dynamiques (dots)
* ✅ Défilement automatique (auto-play)
* ✅ Pause au survol (hover)
* ✅ Support du swipe tactile (mobile)
* ✅ Animation fluide avec `translateX`
* ✅ Slider infini (loop)
* ✅ Lazy loading des images
* ✅ Responsive design

---

## 🧱 Technologies utilisées

* **HTML5**
* **CSS3 (Flexbox)**
* **JavaScript (ES6+)**
